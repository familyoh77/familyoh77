# 사마귀 Wart, Verruca Vulgaris

![image](.gitbook/assets/80692e88-eae1-4712-914f-cedcd89537eb.JPG)

## 일반 사항

* 전염 : 피부 접촉; 긁음에 의하여 autoinoculation 가능
* 호발 : 소아, 젊은 연령층
* 증상 : 고르지 않은 표면(cauliflower 모양); 다른 증상은 없거나 약함(plantar- 통증; anogenital-

가려움; periungual- 건조, 갈라짐, 과각화)

*   경과 : 흔히 자연 소실되지만 잘 치료되지 않거나 재발할 수 있음; 치료로 재발을 막을 수 없음

    •자연 소실률 : 3개월 내 20%, 2년 내 50\~60%

    •치료에 의한 소실률 : 3개월 내 60\~70%

    •자연 소실률 저하 인자 : 생식기 사마귀, 면역저하자, 성인

## 원인

* 원인균 : 사람유두종바이러스(Human papilloma virus)

### 위험 인자

* 손상된 피부, 물에 의해 부드러워진 피부
* 임신, 밀접 접촉 생활(예: 학교), 피부 손상 위험이 많은 직업(예: 날고기/생선 취급자)
* 만성 피부 질환(예: 아토피 피부염), 면역저하, 이전 사마귀 감염

## 종류 및 특징

### 보통 사마귀 (Common wart, Verruca vulgaris)

* 전체 사마귀의 70% 차지
* 모양 : 단단하고 거친 표면의 구진 또는 결절
* 호발 부위 : 사지
* 경과 : ⅔가 2년 내 자연 소실

### 편평 사마귀 (Flat or plane wart, Verruca plana)

* 모양 : 지름 1\~4 ㎜의 작은, 평평하거나 약간 융기, 분홍 또는 피부색, 약간의 비늘, 벨벳 같은 사마귀 같지 않은 표면
* 호발 부위 : 얼굴, 팔, 다리
* 국소 외상(예: 면도)을 따라 선형 분포

### 발바닥 사마귀 (Plantar wart, Verruca plantaris)

* 피부 내부에 위치
* 모양 : 쌀알 모양 구진으로 시작 → 두꺼운 keratin의 collar가 있는 keratotic surface로 진행
* 주의 : 발바닥의 melanoma가 발바닥 사마귀와 유사함

### 생식기 사마귀 (Genital wart)

* 모양 : 작은 유두종으로 시작 → 큰 fungating lesion 형성; 피부색\~갈색 구진 또는 결절
*   부위

    •여성 : 대음순, 회음, 질 점막, 요도, 항문, 항문 주위 피부 침범

    •남성 : 음경체, 음낭, 항문 주위 또는 요도의 관상 고랑(coronal sulcus)에서 시작

## 진단

* 과각화 부분을 깎으면 균일한 붉은색\~검은색 점상 출혈점들(thrombosed capillaries)이 보임

### 검사

* 보통 필요 없음
* 조직 검사(암 감별을 위하여 시행), 전자 현미경, viral DNA 검사

### 감별

* Corn : thrombosed capillaries가 없음 (☞ p.969)
*   암 : squamous cell carcinoma, amelanotic melanoma; 사마귀 치료에 반응하지 않으며 irregular growth 또는 ulceration이

    있으면 의심

***

## Management

### 치료 방침

*   1차 선택 : 국소 salicylic acid &/or 냉동 치료

    •병용 요법으로 치료율을 높일 수 있음 : 냉동 치료 사이에 salicylic acid, 냉동 + 면역 요법
* 2차 선택 : 면역 치료, 5-FU, 레이저
* 생식기 사마귀 : 냉동 치료, podophyllum resin, imiquimod, 레이저, 수술 요법

## 치료법

#### Salicylic acid

* 작용 : 각질 용해
*   용법 : qd(취침 전) ×3~~4개월(사마귀가 제거된 후 1~~2주간 추가 치료) (비보험)

    •따듯한 물에 5\~10분간 soaking 후 손톱 줄이나 부석으로 hyperkeratotic debris를 제거

    → 건조 시킨 후 도포하고 거즈 등으로 가림(가림을 할 경우 격일로 도포할 수 있음)

    • salicylic acid 농도 : 발바닥 사마귀- 25~~50%; 얼굴 편평 사마귀- 2~~5% \[두오필름 겔]\(salicylic acid 16.7%/lactic acid 16.7%)

    •40% salicylic acid plaster : 2\~5일간 유지 후 제거 \[티눈고]
* 치료율 : 수 주간 적용 시 60%\~80%
* 부작용 : 피부 자극(심하면 도포 빈도를 줄임)
*   주의/금기 : 얼굴/생식기/항문 부위, 털이 있는 병변, 홍반이나 특이한 색 병변, 개방창, 감염, 신경병증 발(약제에 의한 피부

    손상의 조기 감지가 어렵고 치유가 지연될 수 있음)

#### 냉동 치료(액체 질소)

* 작용 : 사마귀 균을 냉동시켜 파괴
*   용법 : 두꺼워진 부분을 벗겨내고 30초(15\~45초)간 적용(병변 주위 약 2 ㎜ 피부까지 냉동시킴),

    two freeze-thaw cycles(얼렸다 녹임을 반복); 2\~4주마다 3개월, 또는 총 6회 시술

    •사마귀 제거 후 1\~2주간 salicylic acid 도포

    •손등의 편평 사마귀는 흉터 가능성이 높으므로 짧은 시간(5\~10초) 적용 및 저농도 salicylic acid(≤17%) 도포
* 치료율 : 60\~80%; 시술자의 능력에 영향 받음
*   부작용 : 시술 시 통증(시술 전 국소 마취제 사용 가능), 물집(거즈로 가림; 보통 4\~7일 내 회복),

    인접 피부로의 사마귀 확장, 흉터, 짙은 피부색 환자에서 영구적인 탈색
* 주의/금기 : 손가락(심한 통증, 신경병증 유발), 손톱 주위(영구적 손톱 손상 유발)

#### 5-fluorouracil (5-FU)

* 작용 : 사마귀 바이러스의 DNA/RNA 합성을 억제
* 도포제 용법 : 두꺼워진 부분을 벗겨내고 도포 및 밀폐, qd~~bid ×4~~12주 \[베루말]\(fluorouracil 0.5%/salicylic acid 10%)(비보험)
* 주사제 용법 : 1회/주 ×4주, 병변 내(진피) 주사
* 치료율 : 50%
* 부작용 : 발적, 부종, 피부 변색, 궤양, 괴사, 흉터
* 주의 : 치료 기간 중 도포 부위의 자외선 노출을 피함 (✽자외선이 피부 자극을 악화시킬 수 있음)

#### 면역 요법

* 작용 : 지연 과민 반응을 유도하여 여러 사마귀를 동시에 치료
*   diphenylcyclopropenone(DPCP)

    •용법 : 상박 내측의 이환되지 않은 부위에 2% 용액을 지름 4 ㎝ 정도 크기로 도포 → 2주 후부터 1\~2주마다 점차 농도를 높여

    가면서 병소에 도포(예: 0.001%, 0.01%, 0.1%, 0.2%, 0.5%, 1%, 2%); 매 도포 후 거즈로 가리며 도포 24\~48시간 후 세척

    •치료율 : \~80%
* squalene acid dibutyl ester(SADBE) : 보관 시 변질되기 쉬움
* dinitrochlorobenzene(DNCB) : mutagen 가능성이 있어 사용을 제한
*   부작용 : 홍반, 가려움, 탈색, 수포, 국소/광범위 습진, 두드러기

    •대처 : 피부 부작용이 심한 경우 세척 후 steroid 도포

#### Cantharidin 0.7% 액

* 작용 : epidermal necrosis를 일으킴 (✽FDA 미승인)
*   용법 : 3\~4주마다 1회 ×\~4회

    •도포 후 테이프로 가림 → 수포 발생(2\~24시간) 후 테이프 제거 및 비누 세척

    •사마귀 제거 후 1\~2주간 salicylic acid 도포
* 치료율 : 80%
* 부작용 : 통증, 물집, 부종

#### Imiquimod

* 작용 : 국소 cytokine induction 유도
*   용법 : 주 3회(격일) ×\~3개월 \[알다라 크림]

    •취침 시 도포 → 6\~10시간 후 세척

> ```
> ✽적정 용법은 불확실하여 1일 2회부터 주 3일 적용까지 다양한 적용 보고가 있음
> ```

* 치료율 : 80%(증거 불충분)
* 부작용 : 국소 염증 반응

#### 테이프 밀폐 요법

* 작용 : 작용 원리는 불확실
*   용법 : 잘 떨어지지 않는 waterproof 테이프를 병변이 밀폐되도록 부착 → 6일 후 제거, 따듯한 물로 10분간 soaking 후

    debris 제거 → 다음 날 다시 부착; 총 \~8회 시행
* 치료율 : 85%; 2회 치료에 반응이 없으면 실패할 가능성이 있음
* 주의/금기 : 당뇨병, 신경병증, 말초동맥병, 자극될 수 있는 피부 상태

#### 레이저 치료

* 치료율 : 45\~90%

### 기타

*   경구 cimetidine : 30~~40 ㎎/㎏/d #2~~4 ×8\~12주 (효과 불확실) \[에취투비]

    •작용(가설) : H2 수용체가 T-suppressor cell에 존재하며 이에 대한 receptor를 차단하면 cellmediated immunity가 증가될 수

    있음
* 온탕 목욕 : 뜨거운 물(42.2℃)에 매일 10\~30분씩 담금 ×6주
* 수술적 제거

## 예방

* 공공장소에서의 공공 용품(예: 슬리퍼)을 통한 전파 주의
* 피부를 벗겨내는 기구(예: 손톱깎이, 굳은살 제거 돌) 공동 사용 주의
* 사마귀가 있는 부위의 손상/긁힘(예: 날면도기 사용) 주의
* HPV 백신 : 생식기 사마귀 및 HPV 관련 암 예방 (☞ p.1127)
