# 자율 신경 기능 장애 Autonomic Dysfunction

![image](../.gitbook/assets/5c1a058b-3836-402f-bdc2-68acd51dc333.JPG)

## 임상 양상

* 기립성 저혈압
* 수면 장애
* 운동에 따른 적절한 심박수 조절 이상 (exercise intolerance)
* 발한 장애(과다 또는 저하), 침 분비 감소, 눈물 분비 감소
* 소화기 증상 : 소화 운동 감소, 식욕 부진, 팽만감, 변비, 설사, 삼킴곤란
* 비뇨기 증상 : 배뇨 지연, 요실금, 불완전 뇨배출, 빈뇨
* 성 기능 증상 : 발기 부전, 사정 장애, 질 건조, 오르가즘 장애
* 시력 증상 : 흐릿함, 동공 반사 저하

### 검사

* heart rate variability test
* Valsalva response test
* sudomotor function test

***

## Management

* 기립성 저혈압 치료 (☞ p.500)
* 소화 장애, 요실금 등 증상 치료
* 기저 질환 치료 : 알코올 남용(☞ p.995), 당뇨병(☞ p.540), 파킨슨병 (☞ p.169)
