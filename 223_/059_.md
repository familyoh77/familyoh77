#### PEx

1. **타진**

* 과다 공명 - 공기 ↑ → 천식 / 기흉
* 둔탁 - 액체 or 고형 → 흉막삼출 / 폐렴 / 무기폐

2. **촉진 (진동감)**

* ↓ or 없음 - 차단 (공기/액체) → / 흉막삼출
* ↑ - 전달 증가 → 폐렴

3. **청진**

* 호흡음 없음 →  기흉 / 삼출
* 천명음 → 천식
* 기관지 호흡음 → 폐렴 / 무기폐

### Red Flags

* 기흉 + 저혈압 → 긴장성 기흉 의심 (응급)
* 삼출 + 호흡곤란 진행 → 배액 고려
* 천식 + silent chest → 임박한 호흡부전

## 평가 검사

* 급성 호흡 곤란 : 최고유속검사(peak-flow), 흉부 X선, 심전도, CBC, pulse oximetry 측정 또는 동맥혈가스분석
* 심근경색 : cardiac isoenzyme, troponin
* 천식 : 흡입성 기관지 확장제를 투여한 후 반응 평가
* COPD, 울혈성 심부전 : 6분보행검사
* 폐색전증 : 특별한 원인이 없는 갑작스런 호흡 곤란 환자들에서 의심; echocardiography 등 고려 (☞ p.20)

{% hint style="info" %}
**6분보행검사** : 현재의 삶의 질, 치료 후 평가 및 사망률 예측 도구; 6분간 걸을 수 있는 최대 거리를 측정하고 ≥350 m이면 기능적 제한이 없는 상태, ＜150 m이면 심한 기능적 제한 상태로 판정
{% endhint %}

## 증상에 따른 감별

* 발열 정도와 일치하지 않는 빠른 호흡 → 폐렴 (특히 소아 폐렴의 첫 징후)
* 곤봉지 → COPD
* 심한 비만 → 폐쇄수면무호흡증, 우측성 심부전, restrictive lung Dz
* 길어지는 호기음 → 기관지 폐쇄 초기
* 흉부 전후 직경이 커짐 → 천식, COPD
