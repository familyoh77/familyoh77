# 코피 Epistaxis

![image](.gitbook/assets/285b06db-e4a4-4ba1-ab7b-160cdd0c2eb0.JPG)

## 일반 사항![image](.gitbook/assets/2a90e344-cd1f-48b0-808f-167aedb9baec.JPG)

* 흔한 부위 : Kiesselbach’s plexus(90\~95% 차지)
* 흔한 연령 : 소아(＜15세), 고령층(＞70세)

## 원인 및 위험 인자

*   외부 충격, 외상 : 코 후빔, 코 문지름, 강하게 코 풀기,

    이물질, 운동
* 건조, 자극 : 찬 공기(겨울), 흡연
* 알레르기 또는 바이러스 비염
* 이물, 비중격 만곡, 비용종, 비강 내 종양
* 비내 steroid의 부정확한 사용
*   혈관/응고 관련 : 항응고제, aspirin, 간/신/심부전, 월경,

    백혈병, 응고 인자 결함, 알코올 남용

> ✽고혈압은 보통 코피의 원인이 되지는 않으나 지혈을

> ```
> 지연시킬 수 있음
> ```

## 진단

### 검사

* 일반적으로 필요 없음; 합병증이 있는 경우 또는 경고 징후에 해당되는 경우 고려
* CBC, PT, PTT, basic metabolic panel
* 비강 내시경

***

## Management

### 지혈 방법

1.  약간 앞으로 기울인 자세로 양쪽 코뼈 아래 부위를 단단히 잡고 10\~15분간

    지속적으로 압박한다. (집게를 사용할 수도 있음)![image](.gitbook/assets/a41e7fcd-c9f0-4a12-b3fd-d71db8bc001f.JPG)

•출혈을 줄이기 위하여 oxymetazoline을 비강에 분사하거나 oxymetazoline 또는

```
epinephrine(1:1000)을 솜에 적셔 적용할 수 있음(혈압 상승, 빈맥 주의)
```

2\. 압박에도 불구하고 출혈이 지속될 때는 비강 Packing을 한다.

• petrolatum 연고 \[바셀린]를 적신 거즈를 비강 바닥부터 층층히 쌓아 올림

• 출혈 질환 또는 항응고제를 복용하는 환자에서는 resorbable packing을 사용;

```
gelatin sponge [Surgifoam], collagen sponge [Gelfoam], polymer foam [Merocel]
```

• packing으로 출혈이 멈추면 48시간 동안 유지함

• 비강 패킹 부작용 : 흡인, dislodged packing, 감염, 코 외상

• 비강 패킹을 하는 경우 이를 제거할 때까지 광범위 항생제(예: amox-clav. 250 ㎎ tid \[오구멘틴],

```
TMP-SMX 1T bid [셉트린])투여(근거는 충분하지 않음)
```

3\. 출혈이 멈추지 않거나 재출혈이 발생하면 blood clot을 제거한 후 anterior rhinoscopy로

```
출혈 부위를 확인한다. (15%에서 재출혈이 발생함)
```

4\. 출혈 부위가 확인되면 topical vasoconstrictors, cautery, 또는 moisturizing or lubricating agent를 적용한다.

* 일률적인 국소 또는 경구 항생제 사용을 지지할만한 근거는 없음

#### Cautery

* Silver nitrate 소작 또는 전기 소작

> ✽Silver nitrate는 축축한 점막에 접촉할 때 화학적 소작을 일으킴

* 출혈점이 관찰되는 경우 1차 치료법으로 적용 가능
* 양쪽 비중격 출혈 시 동시에 시술하지는 않음(비중격 천공 위험이 있음)

### 수술

* 보존적 치료에도 불구하고 심한 코피가 발생하는 경우 고려
* ligation, embolization

### 코피를 악화시킬 수 있는 기저 원인 및 관리 방법

*   항혈소판제(예: aspirin, clopidogrel, NSAID) : 가능한 한 약물 투여 중단(중단 후 최대 7\~10일까지 영향 지속);

    필요시 혈소판 수혈
*   항응고제(예: warfarin, enoxaparin, heparin) : 가능한 한 약물 투여 중단(중단 후 영향 지속 기간은 가변적);

    warfarin에 대하여 FFP((fresh frozen plasma), Vit K 투여 고려
* 허브(예: 생강, 은행, 인삼) : 투여 중단(중단후 영향 지속 기간은 가변적)
* thrombocytopenia : 혈소판 수혈
* 간경변증 : PT/INR 모니터링, 급성 출혈에 대하여 FFP 투여 고려
* 신부전, 요독증 : desmopressin 투여, 혈액 투석 고려
* Von Willebrand disease : desmopressin 투여, factor VIII replacement 고려
* 혈우병 : factor replacement 고려

## 예방

* 코피 조절 후 수일간 과로나 격렬한 운동을 피함
* 뜨겁거나 매운 음식 회피, 흡연을 피함
* 비염 관리
* 코 후빔 등 코 손상을 피함
* 식염수 비강 세척(1일 2회) (☞ p.240), 실내 습도 유지(가습기 사용 고려)
*   바셀린 또는 항균 연고 도포 : 성냥 머리 정도 크기로 비강 외측벽(nasal alar 내측)에 바르고 코를 가볍게 쥐어

    비강 내측 벽에 도포되도록 함; mupirocin 1일 1회 \[에스로반]
