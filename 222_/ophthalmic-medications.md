# 안과계 약제 Ophthalmic Medications

## <mark style="color:green;">일반 사항</mark>

* 안과계 약제는 점안액(eye drops), 안연고(ointment), 안젤(gel), 경구제로 구분됨
* 1차 진료에서 처방하는 안약제의 대부분은 점안액·안연고이며, 전신 흡수에 의한 부작용에 주의가 필요함
* 안과 처치 없이 1차 진료에서 처방 가능한 범위와, 안과 의뢰 후 전문의 처방이 원칙인 약제를 구분하여 인지하는 것이 중요함

### <mark style="color:orange;">안약 공통 사용법</mark>

#### <mark style="color:$primary;">점안 방법</mark>

1. 약병을 확인한다.
2. 손을 씻는다.
3. 병을 열기 전에 몇 번 흔든다.
4. 고개를 뒤로 젖힌 후 한 손가락으로 아래 눈꺼풀을 아래로 당긴다.
5. 약병 끝이 눈에 닿지 않는 높이에서 안구와 눈꺼풀 사이에 떨어뜨린다.
6. 점안 후 수 초 동안 눈을 살짝 감고 있는다(반복적인 깜박임은 피함).
7. 눈 안쪽 코 옆(눈물점, 비루관 입구)을 손가락으로 1~2분간 가볍게 눌러 준다. — 약물이 비루관을 통해 코점막으로 흘러들어가 전신 흡수되는 것을 차단하는 조작이다(nasolacrimal occlusion). 베타차단제 안약의 서맥·기관지경련, 스테로이드 점안액의 전신 부작용 등을 줄이는 데 특히 중요하다.
8. 손을 씻는다.

#### <mark style="color:$primary;">공통 주의사항</mark>

* **개봉 후 사용 기한** : 다회 사용 점안액은 개봉 후 4주(1개월) 이내 사용
* **무방부제 제제 선택 기준** : 1일 5회(미국안과학회)~7회(NHS) 이상 지속 사용, 콘택트렌즈 착용자
* **투여량** : 아래 눈꺼풀 안쪽 결막낭에 점안액 1~2방울, 안연고 0.5 인치(약 0.5 ㎝)
* **복수 점안 간격** : 서로 다른 성분의 안약을 함께 사용할 경우 3~5분 이상 간격
* **점안 순서** : 점안액 먼저, 안연고는 마지막(점도가 높아 다른 약 흡수를 방해함)
* **콘택트렌즈** : 점안 전 렌즈 제거 → 점안 후 15분 뒤 재착용 (방부제 함유 제품)
* **현탁액(Suspension) 제제** : 유효 성분이 가라앉아 있으므로 점안 전 반드시 충분히 흔들어서 사용. 해당 약제 : prednisolone acetate, rebamipide <mark style="color:blue;">\[레바아이·레바케이]</mark>, cyclosporine <mark style="color:blue;">\[레스타시스]</mark> 등

## <mark style="color:green;">인공눈물 (Artificial Tears)</mark>

* **적응증** : 안구건조증, 경미한 자극성 충혈, 바이러스 결막염 보조치료, 알레르기 결막염 보조
* **원칙** : 하루 6~7회 이상 사용 또는 콘택트렌즈 착용 시 **무방부제 제품** 사용
* 점안 시 한 방울로 충분하며, 두 방울 이상은 눈물 넘침으로 오히려 손실됨
* **보험 주의** ([건강보험 급여기준 확인](https://www.hira.or.kr/rc/insu/insuadtcrtr/InsuAdtCrtrPopup.do?mtgHmeDd=20241201&sno=3&mtgMtrRegSno=0013))

* hyaluronate(히알루론산) : 점탄성·수분 보유 기전
  * 방부제 함유 병 제품 : <mark style="color:blue;">\[라큐아 점안액]</mark>
  * 무방부제 단회용 : <mark style="color:blue;">\[히알루 미니]</mark>, <mark style="color:blue;">\[히아레인 미니]</mark>

#### <mark style="color:$primary;">점성 유지</mark>

* carboxymethylcellulose (CMC) <mark style="color:blue;">\[리프레쉬 티어즈]</mark>, <mark style="color:blue;">\[인프레쉬 플러스]</mark>
* hydroxypropyl methylcellulose <mark style="color:blue;">\[아티어]</mark>
* polyvinylpyrrolidone <mark style="color:blue;">\[옵타젠트]</mark>

#### <mark style="color:$primary;">표면 장력 완화</mark>

* polyethylene glycol <mark style="color:blue;">\[시스탄]</mark>

#### <mark style="color:$primary;">지질층 보충 / 증발 방지 — EDE/MGD에 특히 적합</mark>

* 지질층 보충 복합제 : 카보머(carbomer) 기반, 글리세린+인지질 혼합 <mark style="color:blue;">\[리포직]</mark>, <mark style="color:blue;">\[시스테인 발란스]</mark>, <mark style="color:blue;">\[아이리스 지질]</mark>
* anhydrous liquid lanolin (취침 전 연고) <mark style="color:blue;">\[듀라티얼즈 안연고]</mark> : 야간 눈 표면 보호
* perfluorohexyloctane (PFHO) <mark style="color:blue;">\[미에보(Miebo)]</mark> : 반불소화 알케인으로 지질층 직접 보충; 수분·방부제 불포함, qid; FDA 승인(2023), 국내 허가 절차 진행 중

#### <mark style="color:$primary;">Mucin Secretagogue — 안구 표면 점액 분비 촉진</mark>

* rebamipide 1회 1적, 1일 4회 <mark style="color:blue;">\[레바아이]</mark>, <mark style="color:blue;">\[레바케이]</mark>
* diquafosol 1회 1적, 1일 6회 <mark style="color:blue;">\[디쿠아이]</mark>

## <mark style="color:green;">스테로이드 점안제</mark>

{% hint style="danger" %}
**스테로이드 점안제 — 1차 진료에서의 금기·주의 원칙**

**절대 금기 (1차 진료 기준)**
- HSV 각막염 (dendritic lesion 확인 시 즉시 의뢰)
- 원인 불명 red eye (특히 통증·시력 저하 동반)
- 세균성 각막염 의심

**상대 금기**
- 바이러스 결막염 (바이러스 배출 연장 가능)
- 진균·아칸토아메바 감염 의심

**원칙 : 1차 진료에서 스테로이드는 안과 진단 이후 사용**
{% endhint %}

### <mark style="color:orange;">성분별 분류</mark>

| 성분명 [상품명] | 역가 | 주요 사용 |
|---|---|---|
| fluorometholone 0.1% <mark style="color:blue;">\[오큐메토론]</mark> | 저~중 | 알레르기 결막염, 안구건조증 염증, 안검염, 콩다래끼 |
| loteprednol etabonate <mark style="color:blue;">\[로테프로]</mark> | 저 | 안구건조증 염증, 안검염 (안압 상승 위험 낮음) |
| rimexolone <mark style="color:blue;">\[벡솔]</mark> | 중 | 안검염 중등증, 콩다래끼 |
| dexamethasone | 고 | 복합제 성분; 단독 처방 시 주의 |
| prednisolone acetate | 고 | 복합제 성분; 전문의 처방 원칙 |

* **사용 기간** : 4회/일 사용 시 2주 미만; 가끔 사용 시 수 주간 허용 (가능한 저역가 선택)
* **장기 사용 부작용** : 안압 상승 → 스테로이드 유발 녹내장, 후낭하 백내장, 감염 악화

| 적응증 | 선택 약제 | 기간 |
|---|---|---|
| 알레르기 결막염 | fluorometholone <mark style="color:blue;">\[오큐메토론]</mark> qid | ≤2주 (증상 조절 후 중단) |
| 안구건조증 (염증 동반) | loteprednol <mark style="color:blue;">\[로테프로]</mark> qid | 1~2주 |
| 안검염 (중등증) | loteprednol or rimexolone qid | 2~3주 이내 |
| 콩다래끼 (초기) | fluorometholone or loteprednol qid | 2~4주 후 재평가 |
| 다래끼 (큰 속다래끼) | tobramycin/dexamethasone <mark style="color:blue;">\[토브라덱스]</mark> | 수일 단기; HSV 배제 후 |

{% hint style="info" %}
**Doxycycline 안과 사용의 특이점**\
안과 질환에서 doxycycline은 항균 목적보다 **항염 및 meibomian gland 기능 개선** 목적으로 저용량 장기 투여하는 경우가 많음. 임부·12세 미만 금기. 식후 충분한 물과 복용, 복용 후 30분 눕지 않도록 지도.
{% endhint %}

{% hint style="warning" %}
**저용량(20~50 ㎎) 처방 시 현실적 용량 조절 가이드**\
국내에는 doxycycline **100 ㎎ 정제**만 유통되는 경우가 대부분임. MGD·안검염·안구건조증에서 저용량 항염 목적으로 처방 시 아래 방법 중 선택:
- **분할 복용** : 100 ㎎ 정제 ½정(50 ㎎) qd — 가장 현실적인 방법 (capsule 제형은 분할 불가)
- **격일 복용** : 100 ㎎ qod — 50 ㎎/일과 유사한 평균 혈중 농도; 복용 일정이 불규칙해질 수 있음
- **20 ㎎ 제형** : 국내 일부 제네릭 제품 유통 확인 후 처방 가능 시 가장 정확함
- **minocycline 50 ㎎으로 대체** : 항염 효과 유사; 분할 없이 정확한 저용량 처방 가능 <mark style="color:blue;">\[미노씬 50 ㎎]</mark>
{% endhint %}

### <mark style="color:$success;">핵심 복약 지도</mark>

> **점안액 올바른 사용법 (공통)**
>
> * 점안 전 반드시 손을 씻으십시오.
> * **현탁액(뿌옇거나 탁한 약액) 제제는 점안 전 반드시 충분히 흔들어서 사용하십시오.** 유효 성분이 가라앉아 있으면 농도가 고르지 않아 효과가 떨어질 수 있습니다. 해당 약제 예시: 레바아이, 레스타시스 등.
> * 아래 눈꺼풀을 살짝 당긴 후, 결막낭(아랫 눈꺼풀 안쪽)에 1~2방울 떨어뜨리십시오. 약병 끝이 눈에 닿지 않도록 주의하십시오.
> * 점안 후 수 초 동안 눈을 살며시 감고, 눈 안쪽(코 옆 눈물점)을 손가락으로 1~2분간 가볍게 눌러 주십시오. 약물이 비루관을 통해 코점막으로 흡수되어 전신으로 들어가는 것을 막는 중요한 조작입니다. 쓴맛, 심박수 변화(베타차단제 계열) 등 전신 부작용을 줄여 줍니다.
> * 여러 종류의 점안액을 함께 사용하는 경우, 점안 간격을 3~5분 이상 두십시오.
> * 점안액을 먼저 넣고, 안연고는 마지막에 넣으십시오.
> * 콘택트렌즈 착용 중이라면 점안 전 렌즈를 먼저 빼고, 점안 후 15분 뒤에 다시 착용하십시오.
> * 개봉 후 4주가 지난 점안액은 오염 우려가 있으므로 사용하지 마십시오.

> **스테로이드 점안액 사용 시 특별 주의사항**
>
> * 의사가 처방한 기간(보통 1~2주) 이상 임의로 연장하지 마십시오. **장기 사용 시 안압 상승 또는 백내장이 발생할 수 있습니다.**
> * 사용 중 눈이 더 불편해지거나 시야가 흐려지면 즉시 내원하십시오.
> * 녹내장이 있는 경우에는 반드시 사전에 의사에게 알리십시오.

> **안과계 경구 항생제 (독시사이클린·미노사이클린) 복용 안내**
>
> * 식후 충분한 물(200 ㎖ 이상)과 함께 복용하십시오. 복용 후 30분 이내에 눕지 마십시오(식도 자극 예방).
> * 복용 중 햇빛·자외선에 노출되면 피부가 민감해질 수 있습니다(광과민 반응). 외출 시 자외선 차단제를 사용하십시오.
> * 임신 중·수유 중 또는 만 12세 미만인 경우에는 복용하지 마십시오.
> * 증상이 호전되어도 처방된 기간을 끝까지 복용하십시오.

### <mark style="color:blue;">환자 안내서</mark>

{% hint style="info" %}
**안과 안약, 이것만 알아두세요**

안과 안약은 종류가 많고 사용법이 비슷해 보이지만, 각각 역할이 다릅니다. 안전하고 효과적으로 사용하기 위한 핵심 사항을 정리했습니다.
{% endhint %}

#### <mark style="color:$primary;">안약의 종류별 역할은 무엇인가요?</mark>

* **인공눈물** : 눈물을 보충해 건조함과 자극을 완화합니다. 가장 안전한 기본 약입니다.
* **항알레르기 안약** : 가려움을 줄여 줍니다. 알레르기 결막염에 사용합니다.
* **항생제 안약** : 세균 감염을 치료합니다. 처방에 따라 정해진 기간만 사용하십시오.
* **스테로이드 안약** : 눈의 염증을 빠르게 가라앉힙니다. **의사의 지시 없이 장기 사용하면 안압 상승·백내장 위험이 있으므로 반드시 처방 기간을 지켜야 합니다.**

#### <mark style="color:$primary;">여러 종류의 안약을 함께 쓸 때는?</mark>

* 각 안약은 **3~5분 간격**을 두고 넣으십시오.
* 점안액을 먼저 넣고, 안연고는 마지막에 넣으십시오(점도가 높은 연고는 다른 약의 흡수를 방해합니다).
* 콘택트렌즈를 착용 중이라면 안약을 넣기 전에 먼저 렌즈를 빼고, 넣은 후 15분이 지나서 다시 착용하십시오.

#### <mark style="color:$primary;">인공눈물, 하루에 몇 번이나 넣어도 되나요?</mark>

* 하루 6~7회 이하라면 일반 제품을 사용해도 됩니다.
* **하루 6~7회 이상** 사용하거나 콘택트렌즈를 착용 중이라면, 방부제가 없는 **단회용(1회용 앰플) 제품**을 선택하십시오. 방부제가 포함된 제품을 자주 오래 사용하면 오히려 눈 표면을 자극할 수 있습니다.

#### <mark style="color:$primary;">이럴 때는 즉시 병원을 방문하세요</mark>

* 안약을 사용 중인데 **시력이 갑자기 저하되거나 흐려지는** 경우
* **심한 눈의 통증** 또는 강한 눈부심이 생기는 경우
* 스테로이드 안약을 처방받아 사용 중인데 **증상이 오히려 악화되거나 시야가 흐려지는** 경우
* 안약을 **처방 기간보다 오래 사용하게 되는** 경우 — 반드시 의사와 상의하십시오
