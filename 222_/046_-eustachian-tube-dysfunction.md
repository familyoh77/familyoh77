# 귀인두관기능부전 Eustachian Tube Dysfunction

## 일반 사항

* 중이 귀인두관의 functional valve 장애로 인하여 적절한 개폐가 이루어지지 않는 상태
* 이관 폐쇄증(확장 기능 장애) 또는 이관 개방증(폐쇄 기능 장애)
* 편측 또는 양측 발생
* 귀인두관의 기능 : 중이 내 환기/압력 조절(중이와 주변 기압의 균형을 맞춤), 비인두 내용물의 역류로부터 보호, 중이 분비물 배출
* 귀인두관의 정상 움직임 : 휴식 시 연골부 근육 작용에 의하여 닫혀 있다가 하품/저작/삼킴 동작 시 열림
* 귀인두관의 지속적인 개방 부전은 심각한 합병증을 유발할 수 있으나, 지속적인 개방으로는 심각한 합병증이 발생하지 않음

## 이관 폐쇄증/이관 확장 기능 장애 (Dilatory dysfunction)

### 원인

* 염증 : 상부 호흡기 감염(아데노이드, 비인두, 코, 부비동의 감염), 알레르기, 자극(예: 담배 연기, 오염 물질), 인후두 역류, 위식도 역류, 호르몬 변화(예: 임신 후기), granulomatous Dz, Samter’s triad, primary ciliary dyskinesia
* 압력 조절 문제 : 잠수, 비행
* 해부학적 문제 : 비인두 종괴, 아데노이드 비대, cholesteatoma, dermoid cyst, cleft palate
* 신경 손상 : 구개열 수술, 삽관(nasogastric, nasal endotracheal tube)
* 퇴행성/대사 질환 : myasthenia gravis
* 비만
* 낮은 사회 경제적 상태

### 임상 양상

* 귀 통증, 귀 충만감/압박감, 청력 감소, 이명
* “귀가 막힘”, 삼키거나 하품을 하면 “펑/딱” 소리 발생
* (중증에서) 어지럼, 불균형

### Red flags!

* 심한 청력 저하
* 심한 통증
* 편측에서 지속
* cholesteatoma
* 지속/재발 중이염

### 합병증

* 귀인두관의 개방 부전 상태가 지속되면 중이에 갇힌 공기가 흡수되어 중이 내 음압 발생 → 중이 내 장액성 삼출액 발생 → 정체된 삼출액에서의 감염/염증 반응 및 악화
* 만성 폐쇄 시 재발성 중이염, 만성 유양돌기염, 진주종 유발

### 진단

* pneumatic otoscopy : 고막 후퇴, 삼출액, 고막 움직임 감소
* Toynbee maneuver : 코와 입을 막고 스스로 바람을 불어 넣으면서(auto insufflate) 고막 관찰; 고막 후퇴, 고막 후상방의 retraction pocket
* audiogram : 전음성 난청
* Weber test : tuning fork(512 ㎐ fork)를 이마에 대었을 때 이환된 귀에서 크게 들림
* Rinne test : tuning fork를 유양돌기에 대었을 때 귀 앞에서보다 크게 들림 (☞ p.236)
* tympanometry : type B 또는 C
* 영상 검사 : 보통 필요 없으며 ＞3개월 지속 또는 합병증 우려 시 고려; CT, fMRI
* 원인 및 다른 문제들에 대한 검사 고려

**ETDQ-7 questionnaire**

* 지난 1개월간 다음 7가지 질문에 대한 증상 발생 정도를 없음(1점)\~극심함(7점) 스케일로 설문
* 질문 : ⓵ 귀 압박감, ⓶ 귀 통증, ⓷ 귀가 막힌, 물속에 있는 느낌, ⓸ 감기 또는 부비동염이 있을 때 귀의 문제, ⓹ 귀에서 “딱딱” 또는 “펑” 소리, ⓺ 귀 울림, ⓻ 잘 들리지 않음
* 판정 : 총점 ＞14.5점 시 귀인두관기능부전으로 진단 (민감도/특이도 100%/100%)

## 이관 개방증/이관 폐쇄 기능 장애 (Patulous dysfunction)

### 원인

* 급격한 체중 감소(＞2.5\~3 ㎏) : 만성 질환, 영양 결핍, 다이어트
* mucosal scarring : 염증, 시술, 방사선 치료
* mucosal atrophy : 알레르기, 만성 위식도 역류
* 신경 근육 이상(근위축) : 뇌졸중, multiple sclerosis, poliomyelitis
* 호르몬 : estrogen↑(임신, 피임제)
* 지속적인 껌 씹기, 코 울혈 제거제, 두개/안면 기형, palatal myoclonus

### 임상 양상

* 자신의 말소리/숨소리가 크게 들림(autophony)
* ear fullness

### 진단

* otoscopy : 건측 코를 막고 숨을 쉴 때 고막이 호흡에 따라 움직임
* supine position에서 일시적으로 증상 완화(고막 움직임이 사라짐)

## Management

## 이관 폐쇄증

#### 국소

* [코 울혈 제거제](051_-allergic-rhinitis.md#a-adrenergic-agonist) : 필요시(예: 잠수, 비행) 비공 당 1회 분무(q12h); 부작용 주의. ≤4일/월로 사용 제한 (비보험); phenylephrine <mark style="color:blue;">\[시네프린]</mark>, naphazoline \[나리스타]\(chlorpheniramine 복합), xylometazoline <mark style="color:blue;">\[오트리빈]</mark>, oxymetazoline <mark style="color:blue;">\[레스피비엔]</mark>&#x20;
* [steroid](051_-allergic-rhinitis.md#undefined-17) : 알레르기비염 시 고려; 이관 폐쇄에 대한 효과는 입증 안 됨

#### 전신

* 코 울혈 제거제 : ear fullness 또는 pressure 등 충만 증상에 대하여 일부 환자에서 도움
  * phenylephrine : 10 ㎎ 필요시(q4h) [\[코미\]](../%EB%B3%B5%ED%95%A9%EC%A0%9C/)
  * pseudoephedrine : 60 ㎎ 필요시(q4\~6h) <mark style="color:blue;">\[슈다페드]</mark>
* steroid : 전신 투여가 일부 동물 연구에서 도움 <mark style="color:blue;">\[소론도]</mark>
* 항생제 : AOM 등 감염이 동반된 경우 고려 (☞ [중이염](048_-otitis-media.md#undefined-16))

#### 기타

* 비행기 여행, 빠른 고도 상승, 잠수 등은 문제가 해결될 때까지 피함
* autoinflation : 비내 감염이 없는 경우 적용 (비내 감염이 있는 경우에는 중이 감염을 촉진할 수 있음)
* 수술 : tympanostomy, eustachian tuboplasty, balloon dilation/tuboplasty

## 이관 개방증

* 안심시킴
* 코 울혈 제거제 사용을 피함
* nasal saline drops(고개를 들고 투입 후 환측으로 머리를 기울임) 또는 saline irrigation
* potassium iodine 8\~10 방울을 주스 한 잔에 섞어서 하루 3번 섭취(점막 비후를 기대)
* 수술 : tympanostomy tube 삽입, 연골 이식, intraluminal catheter placement, E-tube 폐쇄술
