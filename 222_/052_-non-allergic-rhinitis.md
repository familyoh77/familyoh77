# 비-알레르기비염 Non-Allergic Rhinitis

## <mark style="color:green;">일반 사항</mark>

* IgE 매개 알레르기 반응 없이 발생하는 만성 비증상군으로, 일부는 점막 호산구성 염증(NARES), 일부는 자율신경·감각신경 과민(vasomotor/idiopathic)이 주된 기전; 알레르기비염(AR)과 달리 피부반응검사 및 혈청 특이 IgE 음성
* 유병률 : 비염 전체의 약 25%; 알레르기비염과 공존하는 혼합형(Mixed Rhinitis)도 흔함
* 비알레르기비염의 공통 기전 : 자극 → nociceptor(TRPV1·TRPA1) 활성화 → 부교감신경 자극 → 점액 분비·혈관 확장
* 알레르기비염(AR)과의 임상적 감별 요점
  * 눈·코·인두 가려움 또는 발작성 재채기가 현저하지 않음
  * 코 막힘 및 후비루가 AR보다 현저함
  * 감염성 비염 외에는 뚜렷한 계절 요인이 없음 (☞ [알레르기비염](051_-allergic-rhinitis.md))
  * ✽ **혼합형 비염(Mixed Rhinitis) 주의** : 실제 임상에서는 알레르기와 비알레르기 요인이 공존하는 혼합형이 단독 비알레르기비염보다 흔하다(비염 환자의 50\~70% 추정). 알레르기 검사 양성이더라도 찬 공기·자극 냄새 등 비특이적 유발 인자에도 반응한다면 혼합형으로 접근해야 하며, INS 단독보다 복합 치료가 필요한 경우가 많음
  * ✽ **국소 알레르기비염(LAR, Local Allergic Rhinitis) 감별** : 전신 IgE 검사(피부반응·혈청)는 음성이지만 코 점막 내에서만 국소적으로 알레르기 반응이 일어나는 상태. 비알레르기비염으로 오인되는 경우가 많으므로, 표준 검사에서 음성이더라도 특정 항원(집먼지진드기·꽃가루 등) 노출 시 증상이 전형적인 알레르기 패턴이라면 LAR를 의심하고 전문의 의뢰를 고려

### <mark style="color:orange;">정상 비반사 (Normal Nasal Reflexes)</mark>

* **체위 반사 (Postural reflex)** : 누운 자세에서 코 막힘 증가; 옆으로 누우면 아래쪽 비강 막힘 증가
* **겨드랑이 반사 (Crutch reflex)** : 겨드랑이 압박 시 동측 코 막힘 발생
* **온도 반사** : 피부의 열·냉 자극 시 재채기 유발
* **광재채기반사 (Photic Sneeze Reflex, 광 반사)** : 밝은 빛 노출 시 재채기 유발; 상염색체 우성 유전, 인구의 약 18\~35%에서 나타남
* **기관지-비 반사 (Bronchonasal reflex)** : 찬 공기 등 코 자극 시 기관지 수축 유발
* **배란기 비염 (Ovulatory rhinitis)** : 배란 전후에 코 막힘 증가 (에스트로겐·프로게스테론 영향)

### <mark style="color:$danger;">🚩 Red Flags!</mark>

<mark style="color:$danger;">**즉각 의뢰/응급실**</mark>

* **편측 코 막힘 + 안면 통증·부종 + 시력 변화** — 비강 또는 부비강 악성 종양, 안와 합병증 가능성
* **후비루에 혈액 혼재 또는 반복 비출혈** — 종양, 혈관 이형성 배제 필요
* **두개내 합병증 의심** : 심한 두통, 고열, 경부 강직
* **고개를 숙이면 쏟아지는 편측성 맑은 콧물** — 뇌척수액 비루(CSF Rhinorrhea) 의심; 고령자에서 외상 없이도 자발성으로 발생 가능하며 노인성 비염의 맑은 콧물과 혼동되기 쉬움. β₂-transferrin 검사 및 즉시 이비인후과·신경외과 의뢰

<mark style="color:$warning;">**당일 또는 조기 재평가·의뢰**</mark>

* **편측 증상 지속** — 이물질, 비용종, 비강내 종양 감별 필요
* **후각 소실 + 비강 딱지 + 악취** — 위축성 비염, 육아종증 배제
* **4주 이상 치료에 반응 없는 코 막힘** — 해부학적 이상, 종양, 육아종성 질환 고려
* **소아에서 편측 농성 분비물** — 비강 이물질

<mark style="color:$info;">**외래 추적**</mark>

* 비내 스테로이드 치료 2\~4주 후 증상 재평가
* 약물 유발성 비염 : 원인 약물 중단 후 수 주 내 호전 확인
* 약물 반동성 비염 : 비내 스테로이드 전환 후 증상 추적

## <mark style="color:green;">종류 및 특성</mark>

### <mark style="color:orange;">감염성 비염 (Infectious Rhinitis)</mark>

* **원인** : 바이러스(대부분), 세균
* **증상 경과** : 목 증상(인후통·가려움) → 코 증상(코 막힘·콧물·재채기) → 기침 순으로 진행; AR에 비해 서서히 발생, 발열·근육통 등 전신 증상 동반 가능
* **기전** : 코 점막 부종, 섬모 운동 장애, 부비강 배출구 폐쇄·점액 정체, O₂ tension 저하
* **치료** (☞ [부비동염](053_-rhinosinusitis-sinusitis.md), [감기](../223_/060_-common-cold.md))

### <mark style="color:orange;">혈관운동성 비염 / 특발성 비염 (Vasomotor Rhinitis / Idiopathic Rhinitis)</mark>

* **증상** : 코 막힘, 콧물; 간혹 재채기; 계절 변화·눈 증상은 없음
* **기전** : trigeminal 감각신경 과민(TRPV1·TRPA1 활성화) 및 부교감신경 과항진 → 점액 분비·점막 부종·재채기 증가
* **유발 인자** : 온도 변화(덥거나 찬 공기), 운동, 자극(냄새·연기·먼지), 습도, 음주, 뜨겁거나 매운 음식, 고령 — 특정 항원 또는 감염과 무관
* **치료** : 비내 스테로이드 (1차), 비내 항히스타민제 (2차)
* ✽ **용어 정리** : 유발 인자가 뚜렷하면 '혈관운동성 비염', 원인이 불분명하면 **'특발성 비염(Idiopathic Rhinitis)'** 으로 구분하는 추세. 혈관운동성 비염과 NARES를 포괄하는 상위 개념으로 **비감염성 비알레르기비염(NINAR)** 이라는 용어도 사용됨

### <mark style="color:orange;">NARES (Nonallergic Rhinitis with Eosinophilia Syndrome)</mark>

* **증상** : 재채기, 수양성 콧물, 코 막힘, 코 가려움; 알레르기비염·혈관운동성 비염보다 증상이 심한 경향
* **원인** : 불확실
* **관련 인자** : 비-IgE 매개 천식, aspirin 과민증(Samter's triad의 일부로 나타날 수 있음)
* **진단** : 피부단자검사(−) & 혈청 특이 IgE(−) 환자에서 비 분비물 eosinophilia (통상 ≥10–20%; 검사법에 따라 기준 차이)
* **치료** : 비내 스테로이드 (1차), 비내 항히스타민제 (2차), 류코트리엔 억제제 (3차), 단기 경구 스테로이드 (4차, 예외적)

### <mark style="color:orange;">미각 비염 (Gustatory Rhinitis)</mark>

* **증상** : 식사(특히 뜨겁거나 매운 음식) 시 코 막힘·콧물
* **치료** : 식사 15\~30분 전 비내 항콜린제 분무

### <mark style="color:orange;">위축성 비염 (Atrophic Rhinitis)</mark>

* **증상** : 만성 코 막힘, 점막 건조, 딱지(crusting), 악취(ozena); 비강이 비정상적으로 넓어짐, 편평상피화생(squamous metaplasia)
* **원인** : 점막선 세포 위축
* **관련 인자** : 노화, 세균 증식(_K. ozaenae_), 반복적인 코·부비강 수술, 방사선 치료
* **치료** : 매일 비강 세척, 보습 연고 도포; 감염 시 항생제 연고 또는 전신 항생제 사용

### <mark style="color:orange;">노인성 비염 (Senile Rhinitis)</mark>

* 고령에서 알레르기 감작 없이 발생하는 비-알레르기성 비염
* **증상** : 맑은 콧물이 주증상(특히 식사 중 gustatory rhinitis); 코 막힘은 상대적으로 경미; 식사 또는 신체 활동 시 악화; 흔히 점막 위축·건조성 비염 동반
* **기전** : 노화에 따른 **부교감신경 우위** → 점액 분비 과다; 콜라겐·탄성 섬유 감소로 비첨(nasal tip)이 처지고 비전정 협착
* **위축성 비염과의 차이** : 위축성 비염은 점막 위축·딱지·악취가 특징인 반면, 노인성 비염은 **점막 위축 없이 과도한 수양성 분비물**이 특징
* ✽ **CSF 비루 감별 주의** : 고령자의 맑은 콧물이라고 해서 benign으로 보면 안 된다. 편측성이거나 고개를 앞으로 숙일 때 증가하면 CSF 비루를 반드시 배제할 것 — 외상 없이도 자발성으로 발생 가능하며 노인성 비염과 혼동되기 쉬움 (☞ Red Flags, β₂-transferrin 검사)
* **치료** : 비내 항콜린제(ipratropium) 1차 — 맑은 콧물 조절에 효과적; 비강 세척 병행; 항히스타민제보다 비내 ipratropium이 더 효과적

### <mark style="color:orange;">직업성 비염 (Occupational Rhinitis)</mark>

* **증상** : 코 막힘, 콧물; 알레르기 및 비알레르기 특성이 혼재
  * 근무지에서 증상 발생 → 퇴근 후 호전이 특징적
* **원인 물질 예시** : 식품 가공업(음식물 단백질), 제화업(접착제), 화학 공장(암모니아), 마트(세정제·향수·찬 공기)
* **치료** : 원인 노출 회피 (1차), 비내 스테로이드, 비내 cromolyn

### <mark style="color:orange;">호르몬성 비염 (Hormonal Rhinitis)</mark>

* **증상** : 코 막힘, 콧물
* **원인** : 임신(특히 임신 말기\~출산 후 2주), 월경 주기, 경구 피임약, 갑상선저하증
* ✽ **임신성 비염** : 임신 여성의 약 20%에서 발생; 비약물 요법(식염수 비강 세척·비강 확장 스트립)을 우선 시도. 약물 치료가 필요하다면 비내 스테로이드는 **2분기 이후** 사용 가능하며, **budesonide(\[아쿠아미스])** 가 임신 중 가장 풍부한 안전성 데이터(FDA 구분류 B등급 상당)를 보유하고 있어 1차 선택 INS로 권고됨; fluticasone·mometasone은 데이터가 상대적으로 제한적이나 용량이 적고 전신 흡수가 미미하여 임상에서 사용됨

### <mark style="color:orange;">약물 유발성 비염 (Drug-induced Rhinitis)</mark>

* **증상** : 코 막힘, 콧물
* **원인 약물** : ACEI, α/β-차단제, reserpine, guanethidine, phentolamine, aspirin, NSAID, chlorpromazine, amitriptyline, alprazolam, PDE5 억제제, 경구 피임약
* **처치** : 원인 약물 중단 후 수 주 내 회복; 심혈관·정신과 약물은 임의 중단 전 처방의와 협의

### <mark style="color:orange;">약물 반동성 비염 (Rhinitis Medicamentosa)</mark>

* **증상** : 심한 코 막힘(비내 울혈 제거제 사용 중단 시 악화되는 반동 현상)
* **원인** : 비내 울혈 제거제(oxymetazoline, xylometazoline 등) 반복 사용 (보통 3\~5일 초과)
* **기전** : α-수용체 하향 조절 → 반동성 혈관 확장
* **치료** : 비내 스테로이드로 전환(1차); 점진적 농도 희석 또는 단계적 단측 중단; 심한 경우 단기 경구 스테로이드 병용
* ✽ 일반 의약품인 비강 분무제를 장기간 자가 사용하는 환자에서 흔히 발생하므로, 처음 처방 시 **3\~5일 이상 연속 사용 금지** 교육이 중요

### <mark style="color:orange;">해부학적 비염 (Anatomic Rhinitis)</mark>

* **원인** : 비중격 만곡, 비갑개 비후, 비용종, 아데노이드 비대
* **특징** : 보통 편측 발생
* **치료** : 원인별 접근 필요
  * **비중격 만곡** → 이비인후과 의뢰 (스테로이드 적응 없음)
  * **비갑개 비후** → 비내 스테로이드 4–6주 시도 후 반응 없으면 이비인후과 의뢰
  * **비용종** → 비내 스테로이드 ± 단기 경구 스테로이드 후 반응 불충분 시 이비인후과 의뢰
  * **아데노이드 비대** → 이비인후과 평가

## <mark style="background-color:$warning;">Management</mark>

### <mark style="color:orange;">치료 방침</mark>

* 유발 인자·유발 약물 회피
* 식염수 비강 세척 (☞ [알레르기비염](051_-allergic-rhinitis.md#undefined-18))
* 약물 치료 : 대부분의 경우 \*\*비내 스테로이드(INS)\*\*가 1차 선택제
* ✽ **비내 항히스타민/스테로이드 복합제** : INS 단독 불충분 또는 중등도 이상의 혼합형 비염에서 azelastine/fluticasone 복합 분무제(\[딜라스틴] 등)가 최근 가이드라인(ARIA)에서 강력 권고됨. **단, 급여 적응증은 알레르기성 비염으로 한정** — 순수 비알레르기비염에는 비급여; 혼합형에 처방 시 알레르기비염 상병명 병기 필요

### <mark style="color:orange;">유형별 치료제 선택</mark>

J31.0 만성 비염

## <mark style="color:purple;">처방례</mark>

> **처방례 1. 혈관운동성 비염 — 비내 스테로이드 단독**
>
> ```
> [아바미스] 나잘 스프레이   각 비강 2 puffs   qd   (아침, 식후)
> → 증상 조절 후 각 비강 1 puff qd로 감량
> ```
>
> _✽ 비내 스테로이드는 증상 완화까지 2\~4주가 소요됨을 환자에게 미리 안내. 분무구를 눈 바깥쪽(비강 외측벽) 방향으로 향하게 하여 비중격 직접 접촉 회피_

> **처방례 2. 혈관운동성 비염 — 비내 스테로이드 + 경구 항히스타민 병용**
>
> ```
> [아바미스] 나잘 스프레이   각 비강 2 puffs   qd
> [씨잘] 5 ㎎/T   1T   qd   저녁
> ```
>
> _✽ 코 막힘이 주증상일 때는 비내 스테로이드 단독으로도 충분한 경우가 많음. 콧물·재채기가 동반될 때 항히스타민제 추가 고려_

> **처방례 3. 비내 분무제 순응도 낮은 환자 — 경구 항히스타민제 단독**
>
> ```
> [씨잘] 5 ㎎/T   1T   qd   저녁
> ```
>
> _✽ 비내 스테로이드 분무에 거부감이 있거나 고령으로 사용법 습득이 어려운 경우. 경구 항히스타민제 단독은 비내 스테로이드보다 효과가 제한적이므로, 가능하면 비내 분무법 재교육 후 INS를 우선 시도. 1세대 항히스타민제(페니라민 등)는 콜린차단 효과로 분비물 억제에 유리하나 고령자에서 진정·인지기능 저하 주의_

> **처방례 4. 미각 비염 — 비내 항콜린제**
>
> ```
> [아트로벤트] 나잘 스프레이 0.03%   각 비강 2 puffs   식사 15~30분 전
> ```
>
> _✽ 식사 직전에 분무하는 것이 핵심. **비강 건조감·작열감**이 INS보다 빈번하게 발생할 수 있음을 미리 안내. 폐쇄각 녹내장 및 전립선비대증 환자에서 전신 항콜린 부작용(안압 상승·배뇨 곤란) 위험 — 고령 남성에서 특히 주의. 전신 흡수량은 미미하나 해당 병력이 있는 경우 의사에게 미리 알리도록 교육_

> **처방례 5. 약물 반동성 비염 — 비내 스테로이드 전환**
>
> ```
> [아바미스] 나잘 스프레이   각 비강 2 puffs   qd
> (기존 울혈 제거 분무제는 즉시 중단 또는 2주에 걸쳐 단계적 중단)
> ```
>
> _✽ 처음 1_~~_2주 내 반동성 코 막힘 악화를 예상하고 환자를 미리 교육. **순응도를 높이는 실용 팁 — '한쪽씩 교대 중단법'** : 양쪽 동시 중단이 어려울 경우, 한쪽 비강의 울혈 제거제를 먼저 중단하고 비내 스테로이드로 전환한 뒤 1_~~_2주 후 반대쪽도 중단하면 반동 증상을 분산시켜 환자의 불편감을 줄일 수 있음. 심한 반동 증상에는 경구 스테로이드 5\~7일 단기 병용 가능 (\[소론도] 5 ㎎/T 3T qd 아침)_

> **처방례 6. 혼합형 비염 또는 중등도 이상 증상 — 비내 항히스타민/스테로이드 복합제**
>
> ```
> [딜라스틴] 나잘 스프레이 (azelastine 137 ㎍ / fluticasone 50 ㎍)
>    각 비강 1 puff   bid   (아침·저녁)
> ```
>
> _✽ **적응 범위** : ① INS 단독 4주 후 반응 불충분 시, ② 중등도 이상 증상을 보이는 혼합형 비염 초기 환자에서 INS 대신 1차 선택지로 고려 가능 — 최근 ARIA 가이드라인은 중등도 이상 NAR에서 복합제의 빠른 효과와 단독 제제 대비 우월성을 강력히 지지함._ _✽ **보험 급여 주의** : \[딜라스틴]의 급여 적응증은 **'중등도\~중증의 알레르기성 비염'** 으로 한정됨. 순수 비알레르기비염에서는 **비급여**. 혼합형 비염(알레르기+비알레르기 공존)에 처방할 경우 **알레르기비염 상병명(J30.1\~J30.4)을 주상병 또는 부상병으로 반드시 기재**해야 급여 인정 가능._

***

### <mark style="color:$success;">핵심 복약 지도</mark>

> **비내 스테로이드 스프레이 올바른 사용법**
>
> * 분무 전 코를 가볍게 푼 후, 분무구를 \*\*눈 바깥쪽 방향(비강 외측벽)\*\*으로 향하게 하고 분사하십시오. 코 가운데 칸막이(비중격)를 향해 분사하면 코피가 생길 수 있습니다.
> * **분무 후 코를 세게 풀지 마십시오** — 약물이 씻겨 나갑니다.
> * 효과가 나타나기까지 **2\~4주**가 걸립니다. 증상이 바로 나아지지 않아도 꾸준히 사용하십시오.
> * 하루 한 번 사용 제제는 **매일 같은 시간대**에 규칙적으로 사용하는 것이 효과적입니다.

> **비내 울혈 제거 분무제(오트리빈, 화이투벤 등) 주의사항**
>
> * 이 약은 **3\~5일 이상 연속으로 사용하지 마십시오.** 장기 사용하면 약을 끊었을 때 코가 더 막히는 반동 현상이 생기고, 이로 인해 약에 의존하게 될 수 있습니다.
> * 의사 처방 없이 장기간 사용 중이셨다면, 반드시 담당 의사에게 알려 주십시오.

> **미각 비염 — 식사 전 분무제 사용법**
>
> * 뜨겁거나 매운 음식을 드시기 **15\~30분 전**에 각 비강에 분무하십시오.
> * 매 식사마다 사용하는 것이 아니라, 증상이 예상되는 식사 전에만 선택적으로 사용할 수 있습니다.

> **언제 다시 병원을 방문해야 하나요?**
>
> * 비내 스테로이드를 4주 이상 사용해도 증상이 전혀 개선되지 않을 때
> * **한쪽 코에서만 증상**이 지속될 때 — 다른 원인이 있을 수 있습니다
> * 콧물에 **피가 섞이거나** 코피가 반복될 때
> * **후각이 갑자기 저하**되었을 때

***

### <mark style="color:blue;">환자 안내서</mark>

{% hint style="info" %}
**비알레르기비염이란 무엇인가요?**

비알레르기비염은 알레르기(IgE) 반응 없이 코 점막에 염증이 생기는 상태로, 온도 변화·냄새·자극 물질·약물·호르몬·감염 등 다양한 원인에 의해 유발됩니다. 알레르기 검사는 정상으로 나오지만, 코 막힘·콧물·후비루 같은 증상은 알레르기비염 못지않게 나타날 수 있습니다.
{% endhint %}

#### <mark style="color:$primary;">비알레르기비염의 주요 유발 요인</mark>

{% hint style="warning" %}
**🔍 일상에서 흔한 유발 요인**

* **날씨·온도 변화** : 찬 공기, 건조한 환경, 급격한 온도 변화
* **자극 물질** : 담배 연기, 향수, 방향제, 매연, 화학 물질 냄새
* **음식** : 뜨겁거나 매운 음식, 알코올
* **약물** : 혈압약(베타차단제·ACEI), 아스피린, 소염진통제, 코막힘 분무제 장기 사용
* **호르몬 변화** : 임신, 월경 주기, 갑상선기능저하증
{% endhint %}

#### <mark style="color:$primary;">생활 속 관리 방법</mark>

{% hint style="success" %}
**✅ 증상 완화를 위한 생활 습관**

1. **유발 요인을 파악하고 피하십시오.** 증상이 심해지는 상황(향수·담배 연기·찬 공기 등)을 메모해 두면 도움이 됩니다.
2. **식염수 비강 세척**을 하루 1\~2회 시행하면 점막을 촉촉하게 유지하고 자극 물질을 씻어낼 수 있습니다.
3. **실내 습도를 40\~60%로 유지**하십시오. 건조한 환경은 코 점막을 자극합니다.
4. **금연하시고, 간접흡연도 피하십시오.**
5. 코막힘 분무제(오트리빈 등)는 **3\~5일을 초과하여 사용하지 마십시오.** 오래 사용하면 약을 끊었을 때 코가 더 심하게 막히게 됩니다.
{% endhint %}

#### <mark style="color:$primary;">이럴 때는 병원을 방문하세요</mark>

* **한쪽 코에서만 증상**이 오래 지속될 때
* 콧물에 **피가 섞이거나** 코피가 자주 날 때
* **냄새를 잘 못 맡게 되었을** 때
* 4주 이상 치료해도 증상이 나아지지 않을 때
* 코와 함께 **눈·귀·얼굴 통증**이 동반될 때

***

**🩺 비알레르기비염 관리 3원칙**

**① 유발 요인을 파악하세요** — 자신의 증상을 악화시키는 상황(찬 공기·향수·매운 음식 등)을 기록해 두면 회피하는 데 도움이 됩니다.

**② 비내 스테로이드는 꾸준히 사용하세요** — 효과가 나타나려면 2\~4주가 걸립니다. 증상이 바로 나아지지 않는다고 중단하지 마십시오.

**③ 코막힘 분무제(오트리빈 등)는 5일 이상 사용하지 마세요** — 장기 사용하면 오히려 코 막힘이 더 심해지는 악순환이 생깁니다.
{% endhint %}
