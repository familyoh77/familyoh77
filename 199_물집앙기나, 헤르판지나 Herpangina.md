# 물집앙기나, 헤르판지나 Herpangina

![image](.gitbook/assets/d739d21e-6939-47a6-94fe-35c652a5b32c.JPG)

## 일반 사항

* 바이러스 감염에 의한 구강 뒷부분의 다발성 궤양을 동반하는 열성 질환
* 원인균 : coxsackievirus A16, coxsackievirus B, enterovirus 71
* 호발 시기 : 늦봄\~가을
* 호발 연령 : 5세 이하(모든 연령에서 발병 가능)
* 잠복기 : 4\~6(14)일
*   전염 : 대변으로 수 주간 장 바이러스 배출 → 손에 장 바이러스 오염 → 음식물 오염 → 입안

    •respiratory droplet(기침, 재채기)에 의해서도 전염 가능
* 경과 : 대부분 자연 치유(7\~10일); 드물게 무균성 뇌막염 등을 유발

## 임상 양상

* 발열 : 갑작스런 다양한 수준의 열(정상\~40℃; 어릴수록 높음)이 단기간(＜5일) 발생
* 식욕 부진, malaise, 보챔; 두통, 요통, 구토, 복통, 설사
* 구강 : 침 흘림, 통증이 있는 특징적 병소

#### 구강 병소

*   발생 부위 : 앞쪽 tonsillar pillar, 연구개, 목젖, 뒤쪽 인두 벽

    •볼 점막, 편도, 구강 앞 부위에는 거의 발생하지 않음
* 발열이 있는 동안 시작
* 형태 : 1~~2 ㎜의 구진 → 수포 → 2~~3일에 걸쳐 주위 홍반을 동반한 3\~4 ㎜의 얕은 궤양으로 진행
* 평균 5개(1\~15개) 정도의 병소
* 심한 통증이 보통 ＜1주 지속

## 진단

### 검사

* 일반적으로는 시행하지 않음
* 약간의 leukocytosis
* 바이러스 배양 : 수포 또는 구강 세척, 대변으로 시행
